    **DispatchCommand**: Accepts a JSON object `{command: string, params: object}`. Maps command name to concrete Command constructor:
    - "transport.play" -> PlayCommand(engine.getTransport())
    - "transport.stop" -> StopCommand(engine.getTransport())
    - "transport.pause" -> PauseCommand(engine.getTransport())
    - "transport.setBpm" -> SetBpmCommand(engine.getTransport(), params["bpm"])
    - "transport.setTimeSignature" -> SetTimeSignatureCommand(engine.getTransport(), params["numerator"], params["denominator"])
    - "transport.setLoopRegion" -> SetLoopRegionCommand(engine.getTransport(), params["startBeat"], params["endBeat"], params["enabled"])
    - "metronome.setEnabled" -> SetMetronomeEnabledCommand(engine.getAudioGraph().getMetronome(), params["enabled"])
    - "metronome.setVolume" -> SetMetronomeVolumeCommand(engine.getAudioGraph().getMetronome(), params["volume"])
    - "master.setVolume" -> SetMasterVolumeCommand(engine.getAudioGraph().getMasterBus(), params["volume"])
    - "parameter.set" -> SetParameterCommand(engine.getParameterRegistry(), params["id"], params["value"])

    Use the established TSFN + Promise pattern. Extract command name and params on the JS thread, then spawn std::thread to construct and dispatch:
    ```cpp
    Napi::Value DispatchCommand(const Napi::CallbackInfo& info) {
        Napi::Env env = info.Env();
        if (info.Length() < 1 || !info[0].IsObject()) {
            Napi::TypeError::New(env, "Expected command object").ThrowAsJavaScriptException();
            return env.Undefined();
        }
        auto cmdObj = info[0].As<Napi::Object>();
        std::string command = cmdObj.Get("command").As<Napi::String>().Utf8Value();
        // Extract params based on command name...
        // Store extracted values as captured locals (not Napi objects - they can't cross threads)

        auto deferred = Napi::Promise::Deferred::New(env);
        auto tsfn = Napi::ThreadSafeFunction::New(env, Napi::Function::New(env, [](const Napi::CallbackInfo&) {}), "DispatchCommand", 0, 1);

        std::thread([deferred, tsfn, command, /* captured params */]() {
            auto& engine = calliope::Engine::getInstance();
            std::unique_ptr<calliope::Command> cmd;
            // Build command based on command string...
            bool success = false;
            if (cmd) {
                success = engine.getCommandDispatcher().dispatch(std::move(cmd));
            }
            tsfn.BlockingCall([deferred, success](Napi::Env env, Napi::Function) {
                deferred.Resolve(Napi::Boolean::New(env, success));
            });
            tsfn.Release();
        }).detach();

        return deferred.Promise();
    }
    ```

    **CRITICAL:** Extract ALL parameter values from the Napi::Object on the calling thread and capture them as plain C++ types (std::string, double, int, bool, float). Do NOT capture Napi::Object or Napi::Value across threads — they are tied to the V8 isolate.

    **SubscribeToEvents**: Accepts a callback function. Creates a persistent ThreadSafeFunction (NOT the one-shot pattern used for Promises). Creates a `BridgeListener` class that implements `CommandDispatcher::Listener`:
    ```cpp
    // File-static or stored in a global for cleanup
    static Napi::ThreadSafeFunction eventTsfn;
    static std::unique_ptr<BridgeListener> bridgeListener;

    class BridgeListener : public calliope::CommandDispatcher::Listener {
    public:
        explicit BridgeListener(Napi::ThreadSafeFunction tsfn) : tsfn_(tsfn) {}

        void commandExecuted(const calliope::CommandEvent& event) override {
            auto name = event.commandName.toStdString();
            auto data = juce::JSON::toString(event.data).toStdString();
            bool isUndo = event.isUndo;
            tsfn_.BlockingCall([name, data, isUndo](Napi::Env env, Napi::Function callback) {
                auto obj = Napi::Object::New(env);
                obj.Set("type", isUndo ? "undo" : "execute");
                obj.Set("command", Napi::String::New(env, name));
                obj.Set("data", Napi::String::New(env, data));
                callback.Call({obj});
            });
        }

        void commandUndone(const calliope::CommandEvent& event) override {
            auto name = event.commandName.toStdString();
            auto data = juce::JSON::toString(event.data).toStdString();
            tsfn_.BlockingCall([name, data](Napi::Env env, Napi::Function callback) {
                auto obj = Napi::Object::New(env);
                obj.Set("type", "undo");
                obj.Set("command", Napi::String::New(env, name));
                obj.Set("data", Napi::String::New(env, data));
                callback.Call({obj});
            });
        }

    private:
        Napi::ThreadSafeFunction tsfn_;
    };
    ```

    For SubscribeToEvents, create the TSFN with the user's callback function (not a dummy), store it, create BridgeListener, register with dispatcher. **Important**: The TSFN initial thread count should be 1, and it should NOT be released immediately (it's persistent, unlike the Promise pattern).

    **UnsubscribeFromEvents**: Removes the listener from the dispatcher, releases the TSFN, cleans up.
