### Accent Reserved For
- Selected note border/glow (2px stroke at `0x6c63ff`)
- Active toolbar toggle state (Snap button ON, Quantize button ON)
- Playhead line (1px vertical)
- Focus ring on form controls (border-color on `:focus`)
- Selection rubber-band border

### Note Colors
Notes inherit their track color from `TRACK_COLORS` palette. Velocity modulates alpha:
- Velocity 127 (max): track color at 100% opacity
- Velocity 100 (default): track color at 78% opacity
- Velocity 64 (mid): track color at 50% opacity
- Velocity 1 (min): track color at 20% opacity
- Formula: `alpha = 0.2 + (velocity / 127) * 0.8`

Source: D-09 specifies "Note color opacity indicates velocity (darker = higher velocity)."

### Piano Keyboard Colors
- White keys: `#2a2a4a` fill, `#3a3a5a` right border
- Black keys: `#1a1a2e` fill (matches background), `#3a3a5a` border
- Hovered key: lighten by 10% (`#3a3a5a` for white, `#2a2a4a` for black)
- Active/pressed key: accent `#6c63ff` at 40% opacity

### Grid Colors
- Bar boundary lines: `#ffffff` at 12% alpha (matches Phase 6 GridLayer)
- Beat subdivision lines: `#ffffff` at 5% alpha (matches Phase 6 GridLayer)
- C-note row highlight: `#ffffff` at 4% alpha (subtle row tint to mark octave boundaries)

### Velocity Lane Colors
- Velocity bars: same track color as notes, alpha 0.8
- Velocity bar hover: same track color at full opacity
- Velocity lane background: `#1e1e36`
- Velocity lane divider: `#3a3a5a` (1px top border)

## Interaction Contract

### Note Drawing (D-05)
- Tool: Default pointer tool (no explicit mode toggle needed)
- Trigger: Click on empty grid cell
- Behavior: Click sets start position, drag horizontally to set length, release creates note
- Length: Snapped to grid resolution (minimum = 1 grid unit)
- Pitch: Row under pointer at pointerDown
- Default velocity: 100 (D-11)
- Visual feedback: Ghost note rectangle at 50% alpha during drag

### Note Selection (D-06)
- Click note: Select (deselect others)
- Shift+Click: Add to selection
- Ctrl+Click (Cmd on Mac): Toggle in selection
- Drag on empty space: Rubber-band selection box
- Ctrl+A: Select all notes in current clip
- Escape: Clear selection

### Note Move (D-07)
- Trigger: Drag selected note body (not edges)
- Behavior: Move all selected notes by delta (pitch + time)
- Snap: Time snaps to grid, pitch snaps to semitone rows
- Visual: Notes follow pointer with 80% alpha during drag
- Constraint: Notes cannot move below pitch 0 or above pitch 127

### Note Resize (D-07)
- Trigger: Drag note left or right edge (6px hit zone from edge)
- Behavior: Resize note length, snapped to grid
- Minimum length: 1 grid resolution unit
- Cursor: `col-resize` when hovering edge

### Note Duplicate (D-08)
- Ctrl+D: Duplicate selected notes, offset right by each note's own length
- Ctrl+C / Ctrl+V: Copy to clipboard, paste at playhead position

### Note Delete
- Delete/Backspace: Remove selected notes
- Right-click context menu: "Delete Note(s)"

### Velocity Editing (D-10, D-12)
- Single note: Drag velocity bar up/down (0-127 range)
- Multi-note: Select multiple notes, drag one bar to scale all proportionally
- Visual feedback: Tooltip showing velocity value (e.g., "Vel: 100") near pointer during drag
- Velocity bars are vertically aligned with their corresponding notes in the grid

### Quantize (D-13)
- Trigger: Toolbar "Quantize" button or Ctrl+Q
- Target: Selected notes only (no-op if nothing selected)
- Behavior: Snap note start positions to nearest grid line at 100% strength
- Visual feedback: Notes visually shift to quantized positions

### Scroll and Zoom (D-05, PR-05)
- Horizontal scroll: Mouse wheel (no modifier) or trackpad horizontal swipe
- Vertical scroll: Mouse wheel with Shift, or trackpad vertical swipe
- Horizontal zoom: Ctrl+wheel adjusts pixelsPerBeat (shared with timeline)
- Vertical zoom: Ctrl+Shift+wheel adjusts note row height 8-32px (D-03)
- Piano keyboard scales with vertical zoom (key height = row height)

### Split Panel (D-01)
- Drag divider: Resize timeline/piano roll split
- Double-click divider: Toggle piano roll collapse/expand
- Min piano roll height: 200px
- Max piano roll height: 60% of viewport height
- Collapsed state: Only toolbar visible (36px), content hidden

### Context Menu
- Right-click on note: Cut, Copy, Duplicate, Delete, "Set Velocity..."
- Right-click on empty grid: Paste, Select All
- Portal-based rendering matching Phase 6 pattern (D-06 in Phase 6 decisions)

## State Contract

### New Zustand Store: `piano-roll-store.ts`

| State Field | Type | Default | Purpose |
|-------------|------|---------|---------|
| `activeClipId` | `string \| null` | `null` | Currently open MIDI clip |

| `panelHeight` | `number` | `300` | Piano roll panel height |
| `clipboard` | `MidiNote[]` | `[]` | Copied notes for paste |

### MidiNote Type

```typescript
interface MidiNote {
  id: string           // crypto.randomUUID()
  pitch: number        // 0-127 MIDI pitch
  startBeat: number    // Position in beats
  lengthBeats: number  // Duration in beats
  velocity: number     // 1-127
}
```

### GridResolution Type Extension (D-15)

```typescript
// Extend existing type with triplet values
type GridResolution =
  | 0.25 | 0.125 | 0.0625 | 0.03125        // straight: 1/4, 1/8, 1/16, 1/32
  | 0.16667 | 0.08333 | 0.04167             // triplet: 1/4T, 1/8T, 1/16T
```
