  <action>
    Update `PianoRollCanvas.tsx` to add the full interaction layer. Follow the TimelineCanvas.tsx drag state machine pattern exactly (per D-04, D-05, D-06, D-07).

    1. **DragMode and DragState types:**
       ```typescript
       type DragMode = 'none' | 'create' | 'select' | 'move' | 'resize-left' | 'resize-right' | 'velocity'
       interface DragState {
         mode: DragMode
         startX: number
         startY: number
         currentX: number
         currentY: number
         noteId?: string
         originals?: Record<string, { startBeat: number; pitch: number }>
       }
       ```

    2. **findNoteAt helper** (mirrors findClipAt from TimelineCanvas):
       - Convert pixel x,y to beat and pitch accounting for scrollX, scrollY, noteRowHeight, 60px keyboard offset
       - Subtract 60px from px for keyboard width before beat calculation
       - Calculate pitch: `127 - Math.floor((py + scrollY) / noteRowHeight)`
       - Iterate notes matching the pitch
       - Check if beat is within note.startBeat to note.startBeat + note.lengthBeats
       - Check edge proximity (6px threshold per UI spec) for resize detection
       - Return `{ note, edge: 'left' | 'right' | 'body' } | null`

    3. **handlePointerDown** (per D-05, D-06, D-07):
       - If right button (button === 2), return (handled by context menu)
       - Calculate px, py from event relative to overlay rect
       - Call findNoteAt(px, py)
       - **If note hit:**
         - Selection logic per D-06:
           - Shift+click: selectNote(id, true) — multi-select
           - Ctrl/Cmd+click: toggleNoteSelection(id)
           - Plain click on unselected note: selectNote(id) — single select
         - Edge detection:
           - 'left': setDrag mode 'resize-left', store noteId
           - 'right': setDrag mode 'resize-right', store noteId
           - 'body': setDrag mode 'move', store originals for all selected notes (startBeat + pitch)
       - **If no hit and py < (viewportHeight - velocityLaneArea):**
         - clearNoteSelection()
         - setDrag mode 'create', startX/startY stored
       - **If no hit and py >= (viewportHeight - velocityLaneArea) and velocityLaneVisible:**
         - Check if click is on a velocity bar (find note whose horizontal range contains the click x)
         - If found: setDrag mode 'velocity', store noteId, ensure note is selected
       - setPointerCapture on overlay element

    4. **handlePointerMove:**
       - If mode 'none': update cursor based on findNoteAt:
         - Edge hit: `col-resize`
         - Body hit: `pointer`
         - No hit (in grid area): `crosshair`
         - No hit (in velocity area): `ns-resize` if over a bar
       - If mode 'create' (per D-05):
         - Calculate snapped start/end beats from startX and currentX
         - Determine pitch from startY (row under initial pointer position)
         - Show ghost note: div positioned at snapped coordinates, 50% alpha, accent border (#6c63ff/20 bg)
         - Minimum drag distance: 4px before ghost appears
       - If mode 'select':
         - Show rubber-band selection box (absolute div, accent border, transparent fill)
         - Calculate intersecting notes and select them
       - If mode 'move' (per D-07):
         - Calculate deltaBeat from (currentX - startX) / pixelsPerBeat
         - Calculate deltaPitch from row difference: Math.round((startY - currentY) / noteRowHeight)
         - Snap deltaBeat to grid if snapEnabled
         - Call moveNotes([...selectedNoteIds], snappedDeltaBeat, deltaPitch)
       - If mode 'resize-left' or 'resize-right':
         - Calculate new edge beat from currentX, snap to grid
         - For resize-right: newLength = snappedEndBeat - note.startBeat, min = gridResolution
         - For resize-left: newStart = snappedBeat, newLength = (note.startBeat + note.lengthBeats) - snappedBeat
         - Call resizeNote(noteId, newLength) and updateNote for startBeat change
       - If mode 'velocity' (per D-10):
         - Calculate new velocity from pointer Y relative to velocity lane bottom
         - newVelocity = Math.max(1, Math.min(127, Math.round((1 - (py - velocityLaneTop) / velocityLaneHeight) * 127)))
         - If multiple notes selected: call scaleSelectedVelocities(draggedNoteId, newVelocity) (per D-12)
         - Else: updateNote(noteId, { velocity: newVelocity })
         - Show tooltip near pointer: "Vel: {value}" (per interaction contract)

    5. **handlePointerUp:**
       - If mode 'create':
         - If drag distance >= 4px:
           - Calculate snapped start beat, pitch, and length
           - Call addNote(pitch, snappedStart, snappedLength) with default velocity 100 (per D-11)
         - If drag distance < 4px and started on empty: treat as deselect (already cleared in pointerDown)
       - If mode 'select': finalize rubber-band selection
       - For all modes: reset drag state, clear ghost, release pointer capture

    6. **handleContextMenu** (per interaction contract):
       - Prevent default
       - Calculate px, py, call findNoteAt
       - **If note hit:** Show context menu with items:
         - "Cut Notes" -> cutNotes()
         - "Copy Notes" -> copyNotes()
         - "Duplicate Notes" -> duplicateNotes()
         - "Delete Notes" -> removeSelectedNotes(), with destructive: true
       - **If no hit (empty grid):** Show context menu with items:
         - "Paste Notes" -> pasteNotes(beat at click position). Disabled if clipboard empty.
         - "Select All Notes" -> selectAllNotes()

    7. **Ghost note preview** (per D-05 interaction contract):
       - Absolute positioned div with `pointer-events-none`, z-index 11
       - Rounded border, accent color border (#6c63ff), bg #6c63ff/20
       - Position and size calculated from snapped beat coordinates + keyboard offset
       - Only visible during 'create' drag mode when distance >= 4px

    8. **Velocity tooltip:**
       - Absolute positioned span near pointer during velocity drag
       - Shows "Vel: {value}" in 10px, bg #252542, border #3a3a5a, rounded, px-2 py-0.5
       - Follows pointer position with slight offset (+12px right, -20px up)

    9. Wire context menu using `useContextMenu()` hook (same as TimelineCanvas pattern):
       - Import ContextMenu component and useContextMenu hook
       - Render `<ContextMenu>` portal inside the overlay area

    10. Update `PianoRollPanel.tsx` if needed to pass track color info to canvas.
