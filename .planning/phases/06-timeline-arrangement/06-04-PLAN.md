  <action>
    1. **ContextMenu.tsx** — Reusable portal-based context menu (per research code example):
       - Props: `items: Array<{ label: string; action: () => void; destructive?: boolean }>`, `position: { x: number; y: number }`, `onClose: () => void`
       - Renders via `createPortal` to `document.body`
       - Styled per UI-SPEC: `bg-[#252542] border-[#3a3a5a] rounded-md shadow-lg`
       - Items: `px-4 py-2 text-[13px] text-[#eeeeee] hover:bg-[#2a2a4a]`
       - Destructive items: `text-[#ef4444]` (per UI-SPEC destructive color)
       - Closes on any click outside (document click listener)
       - z-index: 50 (above playhead and canvas)

    2. **use-context-menu.ts** — Hook managing context menu state:
       - Returns: `{ visible, position, items, show(event, items), close() }`
       - `show`: prevents default, records mouse position, sets items and visible
       - `close`: hides menu

    3. **use-engine-sync.ts** — Hook syncing engine events to Zustand (per D-10):
       - On mount: calls `window.calliope?.onCommandEvent(callback)`
       - Callback parses event data and updates store via `setTransportState` for transport events
       - On unmount: calls `window.calliope?.removeCommandEventListener()`
       - Gracefully handles missing `window.calliope` (development without engine)

    4. **use-keyboard-shortcuts.ts** — Global keyboard shortcut handler (per UI-SPEC keyboard shortcuts):
       - Registers `keydown` listener on `document`
       - Shortcut map:
         - `Delete` / `Backspace`: delete all selected clips via `deleteClip` for each selectedClipId
         - `Ctrl/Cmd + A`: select all clips — `selectClipsInRect(Object.keys(clips))`
         - `Escape`: `clearSelection()`
         - `Ctrl/Cmd + Z`: `window.calliope?.commandUndo()`
         - `Ctrl/Cmd + Shift + Z`: `window.calliope?.commandRedo()`
         - `Ctrl/Cmd + D`: duplicate each selected clip via `duplicateClip`
         - `Space`: toggle play/pause — `window.calliope?.transportPlay()` or `transportStop()` based on `isPlaying`
         - `S` (no modifier, timeline focused): `toggleSnap()`
         - `Ctrl/Cmd + =`: zoom in (increase pixelsPerBeat by 1.5x, clamp to 192)
         - `Ctrl/Cmd + -`: zoom out (decrease pixelsPerBeat by 1.5x, clamp to 6)
       - Prevent default for handled shortcuts
       - Ignore shortcuts when user is typing in an input/textarea

    5. **Update TimelineCanvas.tsx** — Add clip containers, selection box, loop region, and interaction handlers inside the viewport:
       - Render `<ClipContainer>` for each track inside the viewport (per D-01)
       - Render `<SelectionBox>` inside viewport
       - Render `<LoopRegion>` inside viewport
       - **Clip creation (per D-05)**: On pointerdown + drag on empty track area:
         - Determine which track row (y / 80px)
         - Convert x to beat via `pixelToBeat`, snap with `snapToBeat`
         - Show ghost clip during drag (a semi-transparent rectangle)
         - On pointerup: if drag distance >= 1 grid unit, call `createClip(trackId, startBeat, lengthBeats, 'midi')`
         - Minimum: 1 grid unit (per UI-SPEC)
       - **Clip selection (per D-06)**: On clip pointerdown:
         - Plain click: `selectClip(clipId)` (deselects others)
         - Shift+click: `selectClip(clipId, true)` (add to selection)
         - Ctrl/Cmd+click: `toggleClipSelection(clipId)` (toggle)
         - Click empty area: `clearSelection()`
       - **Rubber-band selection (per D-06)**: On pointerdown on empty area + drag:
         - Track SelectionBox start/end coordinates
         - On pointerup: determine which clips intersect the selection rectangle
         - Call `selectClipsInRect(intersectingClipIds)`
       - **Clip move (per UI-SPEC Clip Move interaction)**: On pointerdown on selected clip + drag:
         - Calculate delta in beats and track rows
         - Snap to grid
         - On pointerup: call `moveClip(clipId, newStartBeat, newTrackId)` for each selected clip
         - Escape cancels move
       - **Clip resize (per D-08)**: On pointerdown near clip edge (within 6px):
         - Change cursor to `col-resize`
         - Drag to resize, snap to grid
         - Minimum clip length: 1 grid unit
         - On pointerup: call `resizeClip(clipId, newStartBeat, newLengthBeats)`
       - **Right-click on clip**: show context menu with items (per UI-SPEC ClipContextMenu):
         - "Delete Clip" (destructive) — `deleteClip(clipId)`
         - "Duplicate Clip" — `duplicateClip(clipId)`
         - "Split at Playhead" — split clip at current playhead beat (creates two clips)

    6. **Update TrackHeader** (in tracks/TrackHeader.tsx) — Add right-click context menu:
       - On contextmenu event: show context menu with items (per D-07, UI-SPEC TrackContextMenu):
         - "Add Track Above" — `addTrack()` with order insertion
         - "Add Track Below" — `addTrack()` with order insertion
         - "Delete Track" (destructive) — show confirmation then `removeTrack(trackId)`
         - "Rename Track" — trigger inline edit mode (input field replaces label, Enter confirms, Escape cancels)
         - "Change Color" — submenu or color swatch grid with 8 colors from TRACK_COLORS
         - "Duplicate Track" — `addTrack(track.name + ' Copy', track.colorIndex)`
       - **Double-click track name**: inline edit mode (per UI-SPEC interaction contract)

    7. **Update TimelineRuler.tsx** — Add loop region creation (per TL-06, UI-SPEC Loop Region interaction):
       - Shift+click+drag on ruler: create loop region
       - Track start/end beat positions during drag
       - On release: call `setLoopRegion({ startBeat, endBeat })`
       - Also dispatch to engine: `window.calliope?.setLoopRegion(startBeat, endBeat, true)`
       - Double-click existing loop region: remove it via `setLoopRegion(null)` and `window.calliope?.setLoopRegion(0, 0, false)`
       - Click on ruler (without Shift): move playhead to clicked beat

    8. **Update TimelineView.tsx** — Wire hooks:
       - Call `useEngineSync()` to start engine event listening
       - Call `useKeyboardShortcuts()` to register global shortcuts
       - Pass context menu state to `<ContextMenu />` rendered at root level
