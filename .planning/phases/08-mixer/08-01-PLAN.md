       - `gainToDb(gain: number): number` — returns `20 * Math.log10(gain)`, or `-Infinity` if gain <= 0

       - `smoothMeter(current: number, target: number, deltaTime: number, attackTime: number, releaseTime: number): number` — exponential smoothing: if target > current use attackTime coefficient, else use releaseTime coefficient. Coefficient = `1 - Math.exp(-deltaTime / time)`. Returns `current + (target - current) * coefficient`.

    Create `app/src/renderer/stores/mixer-store.ts` as a Zustand store with `create<MixerState & MixerActions>()`:

    **State fields:**
    ```typescript
    interface MixerState {
      panelVisible: boolean          // default false
      panelHeight: number            // default 300
      meterLevels: Record<string, { left: number; right: number }>   // per-track RMS
      meterPeaks: Record<string, { left: number; right: number }>    // per-track peak hold
      masterMeterLevel: { left: number; right: number }              // master RMS
      masterMeterPeak: { left: number; right: number }               // master peak hold
      effectChains: Record<string, EffectSlot[]>                     // per-track effects
      masterEffectChain: EffectSlot[]                                // master effects
    }
    ```

    **Actions:**
    - `togglePanel()` — flip panelVisible
    - `setPanelHeight(h: number)` — set panelHeight
    - `updateMeterLevels(data: Record<string, MeterLevel>)` — for each key in data: if key === "master", update masterMeterLevel/masterMeterPeak; else update meterLevels[key] and meterPeaks[key]. Use rmsLeft/rmsRight for levels, peakLeft/peakRight for peaks.
    - `addEffect(trackId: string, effectType: EffectSlot['effectType'])` — look up label from EFFECT_TYPES, push new EffectSlot with `id: crypto.randomUUID()`, `bypassed: false` to effectChains[trackId] (create array if missing)
    - `removeEffect(trackId: string, slotId: string)` — filter out slot from effectChains[trackId]
    - `toggleEffectBypass(trackId: string, slotId: string)` — flip bypassed on matching slot
    - `reorderEffect(trackId: string, fromIndex: number, toIndex: number)` — splice the slot from fromIndex to toIndex within effectChains[trackId]
    - `addMasterEffect(effectType)` — same as addEffect but on masterEffectChain
    - `removeMasterEffect(slotId)` — same as removeEffect but on masterEffectChain
    - `toggleMasterEffectBypass(slotId)` — same pattern on masterEffectChain
    - `reorderMasterEffect(fromIndex, toIndex)` — same pattern on masterEffectChain
