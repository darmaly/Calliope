## Implementation Decisions

### Merge Regression Fixes
- Restore getMeterLevels, setTrackVolume, setTrackPan to preload/index.ts
- Restore engine:meter:getLevels IPC handler in main/index.ts
- Restore getMeterLevels in native-bridge.ts NativeAddon interface
- These are exact restorations of code that existed at commit e891469 but was dropped

### Engine Initialization
- Add initialiseEngine(44100, 512) call in App.tsx useEffect mount
- Must run before any transport/instrument/effect operations

### Dirty Tracking Initialization
- Call initProjectDirtyTracking() in App.tsx useEffect with cleanup on unmount
- Enables reactive window title updates and autosave notification propagation

### Engine Clip Support
- Add clip concept to C++ engine so timeline clips are managed at the engine level
- Bridge clip creation/deletion/modification to allow testing of actual audio playback with clips
- This enables the user to test the full DAW pipeline: create tracks → add clips → write MIDI → hear playback

### Claude's Discretion
- C++ clip data structure design
- Bridge function signatures for clip operations
- Integration approach with existing AudioGraph routing
