<objective>
Wire all three instruments (PolySynth, BassSynth, DrumMachine) into the AudioGraph, register all parameters in ParameterRegistry, create instrument-specific commands (noteOn, noteOff, loadSample), extend the bridge with instrument command dispatch, and add IPC handlers for renderer access.

Purpose: This completes INST-04 by making every instrument parameter controllable via the command dispatcher, and makes instruments playable from the JS/renderer layer. Without this wiring, instruments exist as isolated C++ classes with no way to use them.

Output: Instruments are graph nodes producing sound, all parameters are dispatcher-controllable, bridge supports note events and sample loading, IPC exposes everything to the renderer.
</objective>
