    **drum_machine.h**: Create `DrumMachineProcessor` subclassing `juce::AudioProcessor`.

    - Constructor: `BusesProperties().withOutput("Output", juce::AudioChannelSet::stereo())`. Create 16 `juce::SamplerVoice` instances (one per pad, allows simultaneous playback). Initialize `juce::AudioFormatManager formatManager_` and call `formatManager_.registerBasicFormats()` in constructor.
    - `acceptsMidi()` returns true. `producesMidi()` returns false.
    - Public atomic parameters:
      - `volume` (float, 0.0-1.0, default 0.8)
    - Constants: `static constexpr int kNumPads = 16`, `static constexpr int kFirstMidiNote = 36` (C1, GM kick drum).
    - Thread-safe MIDI injection: same SpinLock + MidiBuffer pattern as synths. `void addMidiEvent(const juce::MidiMessage& msg)`.
    - Sample loading: `bool loadSample(int padIndex, const juce::File& file)`.
      - Validate padIndex (0-15). Validate file exists.
      - Use `formatManager_.createReaderFor(file)` to get `AudioFormatReader*`. Return false if null.
      - Calculate MIDI note = kFirstMidiNote + padIndex.
      - Create `juce::BigInteger noteRange; noteRange.setBit(midiNote)`.
      - Remove any existing SamplerSound for this note range first (iterate synth_ sounds, find matching note).
      - Add new `juce::SamplerSound(file.getFileNameWithoutExtension(), *reader, noteRange, midiNote, 0.001, 0.01, 30.0)`.
      - `delete reader` after SamplerSound construction (SamplerSound copies the audio data).
      - Store pad name in `std::array<juce::String, kNumPads> padNames_`.
      - Return true.
    - `void clearSample(int padIndex)`: Remove the SamplerSound for the given pad.
    - `juce::String getSampleName(int padIndex) const`: Return padNames_[padIndex].
    - `std::vector<juce::String> getSampleNames() const`: Return all pad names.
    - `processBlock()`: Merge pending MIDI (SpinLock), clear buffer, `synth_.renderNextBlock()`, apply volume.
    - `prepareToPlay()`: `synth_.setCurrentPlaybackSampleRate(sampleRate)`.
    - All AudioProcessor stubs (same as MetronomeProcessor pattern).

    ```cpp
    auto file = tempFile.getFile();
    {
        juce::WavAudioFormat wavFormat;
        auto* writer = wavFormat.createWriterFor(
            new juce::FileOutputStream(file), 44100.0, 1, 16, {}, 0);
        if (writer) {
            juce::AudioBuffer<float> testBuffer(1, 4410); // 100ms at 44.1kHz
            for (int i = 0; i < 4410; ++i)
                testBuffer.setSample(0, i, std::sin(2.0 * M_PI * 440.0 * i / 44100.0));
            writer->writeFromAudioSampleBuffer(testBuffer, 0, 4410);
            delete writer;
        }
    }
    ```
