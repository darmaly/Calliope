    **poly_synth.h**: Create `PolySynthProcessor` subclassing `juce::AudioProcessor`.
    - Constructor: `BusesProperties().withOutput("Output", juce::AudioChannelSet::stereo())`. Add 16 `PolySynthVoice` instances to internal `juce::Synthesiser synth_`. Add one `PolySynthSound`.
    - `acceptsMidi()` returns true. `producesMidi()` returns false.
    - Public atomic parameters (all `std::atomic` with `memory_order_relaxed` access):
      - `osc1Waveform` (int, 0=saw, 1=square, 2=sine, 3=triangle, default 0)
      - `osc2Waveform` (int, default 2 for sine)
      - `oscMix` (float, 0.0-1.0, default 0.5, balance between osc1 and osc2)
      - `osc2Detune` (float, -100 to +100 cents, default 0)
      - `filterCutoff` (float, 20.0-20000.0 Hz, default 5000.0)
      - `filterResonance` (float, 0.0-1.0, default 0.1)
      - `filterEnvAmount` (float, 0.0-1.0, default 0.3)
      - `ampAttack` (float, 0.001-5.0 seconds, default 0.01)
      - `ampDecay` (float, 0.001-5.0 seconds, default 0.1)
      - `ampSustain` (float, 0.0-1.0, default 0.8)
      - `ampRelease` (float, 0.001-10.0 seconds, default 0.3)
      - `filterAttack` (float, default 0.01)
      - `filterDecay` (float, default 0.2)
      - `filterSustain` (float, default 0.5)
      - `filterRelease` (float, default 0.3)
      - `lfoRate` (float, 0.1-20.0 Hz, default 1.0)
      - `lfoDepth` (float, 0.0-1.0, default 0.0)
      - `lfoTarget` (int, 0=pitch, 1=filter, 2=amp, default 1)
      - `masterGain` (float, 0.0-1.0, default 0.7)
    - Thread-safe MIDI injection: `void addMidiEvent(const juce::MidiMessage& msg)` uses `juce::SpinLock midiLock_` + `juce::MidiBuffer pendingMidi_`.
    - `processBlock()`: Lock SpinLock, merge `pendingMidi_` into the incoming `midi` buffer, clear `pendingMidi_`, unlock. Then `buffer.clear()`, `synth_.renderNextBlock(buffer, midi, 0, buffer.getNumSamples())`, apply `masterGain` to all samples.
    - `prepareToPlay()`: `synth_.setCurrentPlaybackSampleRate(sampleRate)`.
    - All required AudioProcessor stubs (same pattern as MetronomeProcessor).

    **poly_synth_voice.h / poly_synth_voice.cpp**: Create `PolySynthVoice` subclassing `juce::SynthesiserVoice`.
    - Constructor takes `PolySynthProcessor& parent`.
    - `canPlaySound()` returns true if sound is `PolySynthSound*`.
    - Private members: `juce::ADSR ampEnvelope_`, `juce::ADSR filterEnvelope_`, per-voice `juce::dsp::LadderFilter<float> filter_`, LFO state (`float lfoPhase_`), two oscillator phases (`double osc1Phase_`, `double osc2Phase_`), `double currentFrequency_`, `float level_`, `double sampleRate_`.
    - `startNote()`: Read ADSR params from parent atomics at note-on time. Set `ampEnvelope_` and `filterEnvelope_` parameters. Call `noteOn()` on both. Calculate `currentFrequency_` from `juce::MidiMessage::getMidiNoteInHertz(midiNoteNumber)`. Store `level_ = velocity`. Reset oscillator phases.
    - `stopNote()`: If `allowTailOff`, call `noteOff()` on envelopes. Else `reset()` + `clearCurrentNote()`.
    - `renderNextBlock()`: Per-sample loop:
      1. Get amp envelope sample. If not active, `clearCurrentNote()` and break.
      2. Read waveform type from `parent_.osc1Waveform` and `parent_.osc2Waveform`.
      3. Generate osc1 sample using PolyBLEP bandlimited waveform (see RESEARCH.md code examples). Apply osc2 with detune.
      4. Mix osc1 and osc2 using `parent_.oscMix`.
      5. Get filter envelope sample. Calculate filter cutoff = base cutoff + (filterEnvAmount * envValue * 10000Hz range).
      6. Apply LFO modulation (if lfoTarget is filter, modulate cutoff; if pitch, modulate frequency; if amp, modulate output level).
      7. Process sample through `filter_` (use `dsp::LadderFilter` in LPF24 mode). Set cutoff per-sample via `filter_.setCutoffFrequencyHz()`.
      8. Apply amp envelope to output.
      9. Write stereo output via `buffer.addSample()`.
    - Implement private `generateOscSample(int waveform, double& phase, double phaseIncrement)` that uses PolyBLEP for saw and square. For sine: `std::sin(2*pi*phase)`. For triangle: `2.0*fabs(2.0*phase - 1.0) - 1.0` with PolyBLEP correction.
    - Implement private `float polyBlep(float t, float dt)` helper (see RESEARCH.md).
    - `prepareToPlay()` equivalent: `isVoiceActive()` is managed by Synthesiser. In the voice's rendering, check `sampleRate_` set from parent's `prepareToPlay()` notification. Actually, use `getSampleRate()` inherited from SynthesiserVoice.
    - Prepare filter in `startNote()`: call `filter_.reset()`, `filter_.prepare({getSampleRate(), 1, 1})` (mono processing per voice, per-sample).

    - Constructor takes `BassSynthProcessor& parent`.
    - Similar to PolySynthVoice but simpler: single main oscillator + sub-oscillator (sine wave one octave below by default).
    - Private members: `juce::ADSR ampEnvelope_`, `juce::ADSR filterEnvelope_`, `juce::dsp::LadderFilter<float> filter_`, `double mainOscPhase_`, `double subOscPhase_`, `double currentFrequency_`, `float level_`.

    - `renderNextBlock()`: Per-sample:
      1. Amp envelope sample. If not active, clearCurrentNote and break.
      2. Generate main oscillator using PolyBLEP (reuse same polyBlep helper, or copy it -- it's a small function).

      5. Filter envelope modulates cutoff (same pattern as PolySynth).
      6. Process through LadderFilter (LPF24 mode, lower default cutoff).
      7. Apply amp envelope.
      8. Write stereo output.
