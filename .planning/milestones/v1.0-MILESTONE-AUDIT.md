## Critical Integration Gaps (P0)

### 1. Merge Regression: Mixer IPC Methods Missing
**Affected:** MIX-01, MIX-04
**Root cause:** Phase 9-01 worktree branched before Phase 8-02 merge. The merge result dropped `getMeterLevels`, `setTrackVolume`, `setTrackPan` from preload and IPC handlers.
**Fix:** Re-add 3 methods to preload/index.ts, restore IPC handler in main/index.ts, restore native-bridge.ts interface entries. ~10 minutes of work.

### 2. Engine Initialization Never Called
**Affected:** ENG-01 (all engine operations)
**Root cause:** App.tsx useEffect calls `getEngineInfo()` but not `initialiseEngine(44100, 512)`.
**Fix:** Add `initialiseEngine` call in App.tsx mount effect. ~2 minutes.

### 3. Dirty Tracking Initialization Missing
**Affected:** PROJ-02
**Root cause:** `initProjectDirtyTracking()` exported from project-store but never called from App.tsx or main.tsx.
**Fix:** Call `initProjectDirtyTracking()` in App.tsx mount effect. ~2 minutes.
